## Вход в систему

1. Перейдите на страницу входа: `/auth/login`
2. Введите учетные данные администратора:
   - **Логин:** `admin`
   - **Пароль:** `admin123`
3. После успешного входа вы будете перенаправлены в админ-панель

**Важно:** После первого входа рекомендуется изменить пароль по умолчанию.

## Дашборд

Дашборд (`/admin/`) - главная страница административной панели, отображающая:

- **Статистику:**
  - Количество пользователей
  - Количество товаров
  - Количество заказов
  - Заказы, ожидающие обработки
  - Количество статей в блоге
  - Количество элементов контента
  - Новые обращения

- **Последние заказы:** Список из 10 последних заказов с возможностью перехода к деталям

## Управление пользователями

### Просмотр списка пользователей

1. Перейдите в раздел **Пользователи** (`/admin/users`)
2. Отображается таблица со всеми пользователями:
   - ID
   - Имя пользователя
   - Email
   - Роль (admin, manager, user)
   - Статус (активен/неактивен)

### Редактирование пользователя

1. В списке пользователей нажмите **Редактировать** рядом с нужным пользователем
2. Доступные поля для редактирования:
   - Имя пользователя
   - Email
   - Полное имя
   - Телефон
   - Адрес
   - Роль (ADMIN, MANAGER, USER)
   - Статус активности
3. Нажмите **Сохранить** для применения изменений

**Примечание:** Изменение роли пользователя влияет на его доступ к функциям системы.

## Управление контентом (CMS)

### Список контента

1. Перейдите в раздел **Контент (CMS)** → **Список контента** (`/admin/content`)
2. Контент сгруппирован по секциям:
   - `blog` - статьи для блога
   - `hero-slides` - контент для главной страницы
   - `about` - контент страницы "О компании"
   - `contact` - контактная информация
   - `content` - прочий контент

### Редактирование контента

1. В списке контента нажмите **Редактировать** рядом с нужным элементом
2. Доступные поля:
   - **Ключ (key)** - уникальный идентификатор
   - **Заголовок (title)** - название элемента
   - **Содержание (content)** - текст или URL изображения
   - **Секция (section)** - принадлежность к разделу сайта
   - **Тип контента** - text, image, html
3. Нажмите **Сохранить**

### Страница About

1. Перейдите в **Контент (CMS)** → **Страница About** (`/admin/about`)
2. Редактируемые элементы:
   - **Основной текст** - описание компании
   - **Статистика:**
     - Лет опыта
     - Количество мастеров
     - Количество стран
   - **Изображения (4 шт.):**
     - Загрузка файла или указание URL
     - Приоритет у загруженного файла

### Страница Contact

1. Перейдите в **Контент (CMS)** → **Страница Contact** (`/admin/contact`)
2. Редактируемые поля:
   - **Адрес**
   - **Телефон**
   - **Email**
   - **Часы работы**

## Управление категориями

### Создание категории

1. Перейдите в **Товары и заказы** → **Категории** (`/admin/categories`)
2. Нажмите **Добавить**
3. Заполните форму:
   - **Название** - отображаемое имя категории
   - **Slug** - URL-дружественный идентификатор
   - **Описание** - описание категории
4. Нажмите **Сохранить**

### Редактирование категории

1. В списке категорий нажмите **Редактировать**
2. Измените нужные поля
3. Нажмите **Сохранить**

---

## Управление товарами

### Создание товара

1. Перейдите в **Товары и заказы** → **Товары** (`/admin/products`)
2. Нажмите **Добавить**
3. Заполните форму:
   - **Название** - название товара
   - **Slug** - URL-дружественный идентификатор
   - **Описание** - полное описание товара
   - **Краткое описание** - краткое описание для карточек
   - **Цена** - цена в рублях
   - **Количество на складе** - остаток товара
   - **Категория** - выбор категории из списка
   - **Изображение:**
     - Загрузка файла (PNG, JPG, JPEG, GIF, WEBP)
     - Или указание URL изображения
     - Приоритет у загруженного файла
   - **Активен** - отображать товар в каталоге
4. Нажмите **Сохранить**

### Редактирование товара

1. В списке товаров нажмите **Редактировать**
2. Измените нужные поля
3. При изменении изображения:
   - Загрузите новый файл или укажите новый URL
   - Старый файл будет автоматически удален
4. Нажмите **Сохранить**

## Управление заказами

### Просмотр списка заказов

1. Перейдите в **Товары и заказы** → **Заказы** (`/admin/orders`)
2. Отображается список всех заказов:
   - ID заказа
   - Пользователь
   - Сумма заказа
   - Статус
   - Дата создания

### Просмотр деталей заказа

1. В списке заказов нажмите **Подробнее**
2. Отображается:
   - Информация о заказе (пользователь, дата, сумма, адрес, телефон)
   - Список товаров в заказе
   - Форма изменения статуса

### Изменение статуса заказа

1. Откройте детали заказа
2. Выберите новый статус из списка:
   - **Ожидает** (pending) - заказ создан, ожидает обработки
   - **В обработке** (processing) - заказ обрабатывается
   - **Отправлен** (shipped) - заказ отправлен клиенту
   - **Доставлен** (delivered) - заказ доставлен
   - **Отменен** (cancelled) - заказ отменен
3. Нажмите **Обновить статус**

## Управление обращениями

### Просмотр обращений

1. Перейдите в раздел **Обращения** (`/admin/messages`)
2. Доступны фильтры:
   - Все обращения
   - Только непрочитанные
3. Отображается:
   - Имя отправителя
   - Email
   - Тема
   - Дата отправки
   - Статус прочтения

### Просмотр деталей обращения

1. В списке обращений нажмите **Подробнее`
2. Отображается полная информация:
   - Имя отправителя
   - Email
   - Телефон (если указан)
   - Тема
   - Сообщение
   - Дата отправки
3. Обращение автоматически помечается как прочитанное

### Удаление обращения

1. В деталях обращения нажмите **Удалить**
2. Подтвердите удаление

## Управление слайдами Hero

### Создание слайда

1. Перейдите в **Контент (CMS)** → **Слайды Hero** (`/admin/hero-slides`)
2. Нажмите **Добавить**
3. Заполните форму:
   - **Заголовок** - основной заголовок слайда
   - **Подзаголовок** - дополнительный текст
   - **Изображение:**
     - Загрузка файла или указание URL
   - **Ссылка** - URL для перехода при клике на слайд
   - **Текст кнопки** - текст на кнопке (если указана ссылка)
   - **Порядок** - порядок отображения (меньше = выше)
   - **Активен** - отображать слайд
4. Нажмите **Сохранить**

### Редактирование слайда

1. В списке слайдов нажмите **Редактировать**
2. Измените нужные поля
3. Нажмите **Сохранить**

### Удаление слайда

1. В списке слайдов нажмите **Удалить`
2. Подтвердите удаление
3. Изображение слайда будет автоматически удалено

## Полезные советы

- **Резервное копирование:** Регулярно создавайте резервные копии базы данных
- **Безопасность:** Используйте сложные пароли и регулярно их меняйте
- **Изображения:** Оптимизируйте изображения перед загрузкой для ускорения загрузки сайта
- **Контент:** Проверяйте контент перед публикацией на наличие ошибок
