## Регистрация и вход

### Регистрация нового аккаунта

1. Перейдите на страницу регистрации: `/auth/register`
2. Заполните форму:
   - **Имя пользователя** - уникальное имя для входа (обязательно)
   - **Email** - ваш email адрес (обязательно)
   - **Пароль** - минимум 6 символов (обязательно)
   - **Подтверждение пароля** - повторите пароль (обязательно)
   - **Полное имя** - ваше имя (необязательно)
   - **Телефон** - ваш номер телефона (необязательно)
3. Нажмите **Зарегистрироваться**
4. После успешной регистрации вы будете перенаправлены на страницу входа

### Вход в систему

1. Перейдите на страницу входа: `/auth/login`
2. Введите ваши учетные данные:
   - **Имя пользователя** или **Email**
   - **Пароль**
3. Опционально: установите галочку **Запомнить меня** для автоматического входа
4. Нажмите **Войти**

### Выход из системы

1. Нажмите на ваше имя пользователя в правом верхнем углу
2. Выберите **Выход**

## Просмотр каталога

### Навигация по каталогу

1. Перейдите в раздел **Каталог** из главного меню
2. Доступны следующие возможности:
   - **Поиск товаров** - введите название в поле поиска
   - **Фильтрация по категориям** - выберите категорию из списка
   - **Просмотр всех товаров** - нажмите кнопку "Все"

### Просмотр товара

1. В каталоге нажмите на карточку товара или его название
2. На странице товара отображается:
   - Изображение товара
   - Название
   - Цена
   - Описание
   - Количество на складе
   - Кнопка добавления в корзину

### Поиск товаров

1. В каталоге введите название товара в поле поиска
2. Нажмите Enter или кнопку поиска
3. Отобразятся товары, соответствующие запросу

## Работа с корзиной

### Добавление товара в корзину

1. Откройте страницу товара
2. Укажите количество (если нужно)
3. Нажмите **Добавить в корзину**
4. Товар будет добавлен в корзину

### Просмотр корзины

1. Нажмите на иконку корзины в главном меню или перейдите в **Корзина**
2. В корзине отображается:
   - Список товаров
   - Количество каждого товара
   - Цена за единицу
   - Общая стоимость товара
   - Общая сумма заказа

### Изменение количества товара в корзине

1. В корзине измените количество в поле рядом с товаром
2. Нажмите **Обновить корзину**
3. Сумма пересчитается автоматически

### Удаление товара из корзины

1. В корзине нажмите **Удалить** рядом с нужным товаром
2. Товар будет удален из корзины

## Оформление заказа

### Переход к оформлению

1. Убедитесь, что в корзине есть товары
2. Нажмите **Оформить заказ** в корзине

### Заполнение данных для доставки

1. На странице оформления заказа заполните форму:
   - **Адрес доставки** - полный адрес доставки (обязательно)
   - **Телефон** - ваш контактный телефон (обязательно)
   - **Комментарий** - дополнительные пожелания (необязательно)
2. Проверьте состав заказа и общую сумму
3. Нажмите **Подтвердить заказ**

### После оформления заказа

1. Вы будете перенаправлены на страницу подтверждения
2. Заказ получит статус **Ожидает** (pending)
3. Вы получите номер заказа для отслеживания
4. Менеджер обработает ваш заказ и изменит его статус

## Просмотр заказов

### Список заказов

1. Нажмите на ваше имя пользователя в правом верхнем углу
2. Выберите **Мои заказы**
3. Отображается список всех ваших заказов:
   - Номер заказа
   - Дата создания
   - Сумма заказа
   - Статус заказа

### Детали заказа

1. В списке заказов нажмите на номер заказа или **Подробнее**
2. Отображается полная информация:
   - **Информация о заказе:**
     - Номер заказа
     - Дата создания
     - Статус
     - Адрес доставки
     - Телефон
   - **Товары в заказе:**
     - Название товара
     - Количество
     - Цена за единицу
     - Итоговая стоимость
   - **Общая сумма заказа**

### Статусы заказов

- **Ожидает** (pending) - заказ создан, ожидает обработки
- **В обработке** (processing) - заказ обрабатывается менеджером
- **Отправлен** (shipped) - заказ отправлен вам
- **Доставлен** (delivered) - заказ доставлен
- **Отменен** (cancelled) - заказ отменен

## Управление профилем

### Просмотр профиля

1. Нажмите на ваше имя пользователя в правом верхнем углу
2. Выберите **Профиль**

### Редактирование профиля

1. В профиле нажмите **Редактировать**
2. Доступны для редактирования:
   - **Полное имя**
   - **Email**
   - **Телефон**
   - **Адрес**
   - **Пароль** (для изменения пароля)
3. Нажмите **Сохранить** для применения изменений

## Доступность

### Переключение режима для слабовидящих

1. На каждой странице сайта в правом верхнем углу находится кнопка с иконкой глаза
2. Нажмите на кнопку для переключения между стандартным и доступным режимами
3. Ваш выбор сохраняется в браузере и будет применяться при следующих посещениях

### Особенности режима для слабовидящих

Режим для слабовидящих соответствует ГОСТ Р 52872-2019 и включает:

- **Высокий контраст** - контрастность не менее 4.5:1
- **Увеличенные шрифты** - минимальный размер 18px
- **Четкие границы** - границы элементов 2-3px
- **Увеличенные отступы** - улучшенная читаемость
- **Упрощенная цветовая схема** - черно-белая палитра
- **Улучшенная навигация** - подчеркнутые ссылки
